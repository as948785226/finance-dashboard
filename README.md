# 尬酒馆财务看板

🍺 股东专属财务看板 - 随时随地查看收支动态

## 功能特点

- 📱 **移动端适配** - 手机/平板/电脑均可访问
- 🔒 **无需飞书账号** - 一键查看，告别频繁登录
- 📊 **实时数据** - 同步飞书多维表格，自动刷新
- 💰 **账户余额** - 每笔记录显示实时余额
- 📅 **按月查看** - 11月至今各月收支一目了然

## 快速部署

### 方式一：本地运行

```bash
# 1. 进入目录
cd finance-dashboard

# 2. 设置飞书 Bot Token
export FEISHU_BOT_TOKEN="your_bot_token"

# 3. 启动服务
npm start

# 4. 访问 http://localhost:3000
```

### 方式二：内网穿透（供外部访问）

```bash
# 使用 ngrok
ngrok http 3000

# 会生成一个公网地址，如 https://xxx.ngrok.io
# 分享给其他股东即可访问
```

### 方式三：部署到云端

推荐使用以下免费平台：
- **Railway** (railway.app) - 免费额度充足
- **Render** (render.com) - 配置简单
- **Heroku** (heroku.com) - 稳定可靠

## 配置说明

### 获取飞书 Bot Token

1. 前往 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用
3. 获取 App ID 和 App Secret
4. 调用 `oauth/v3/tenant_access_token/internal` 获取 Token

### 环境变量

| 变量名 | 说明 | 示例 |
|--------|------|------|
| FEISHU_BOT_TOKEN | 飞书 Bot Token | tgnxxxxxx |
| BITABLE_TOKEN | 多维表格 Token | EprMbxxx |

## 分享给股东

1. 部署服务后获取公网地址
2. 将链接发送给股东
3. 股东无需登录，直接查看！

## 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JS
- **后端**: Node.js + Express
- **数据源**: 飞书多维表格 API

## 注意事项

- 确保飞书应用的权限已开通「多维表格读取」
- 建议定期备份数据
- 如需修改表格结构，需同步更新代码
